# GachaAddiction

[English](README_EN.md) | [中文](README.md)

A visual gacha/loot display mod for Minecraft NeoForge 1.21.1.

When opening loot containers, GachaAddiction replaces the standard inventory screen with an animated slot-machine style reel interface that reveals the loot in a dramatic presentation.

You can press SPACE to skip the current reel animation or hold ESC to dismiss all animations.

## Features

- **Gacha-style reveal** — Loot items are displayed in a spinning reel animation with decoy items
- **Supports any loot table** — Works with all vanilla loot tables and data pack additions
- **Rarity colors** — Item rarities are highlighted with colored selection indicators
- **Lootr compatible** — Fully compatible with Lootr lootable chests
- **RarityCore support** — Integrates with RarityCore for custom rarity colors if installed

## Dependencies

- **Required**: NeoForge 21.1.200+ (both server and client)
- **Recommended**: RarityCore, ColorTooltips
